# DFX\_OUT

## Function

Encapsulates the output parameters of the L2 first-phase API aclnn_Xxx_GetWorkspaceSize.

## Prototype

```cpp
DFX_OUT(...)
```

## Parameters

| Parameter| Input/Output| Description|
| --- | --- | --- |
| ... | Input| Output parameters of the L2 first-phase API on the host. It is a variable-length parameter.|

## Restrictions

None

## Examples

```cpp
// Output parameter of the Add operator: out
DFX_OUT(out);
```
