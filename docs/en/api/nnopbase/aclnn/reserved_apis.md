# Reserved APIs

This section lists the reserved common APIs, which may be changed or deprecated in later versions. Therefore, you are advised not to pay attention to or use these APIs.

**Table 1** Reserved APIs

| API| Description|
| --- | --- |
| AclSetInputTensorAddr | Has the same function as [aclSetInputTensorAddr](aclSetInputTensorAddr.md).|
| AclSetOutputTensorAddr | Has the same function as [aclSetOutputTensorAddr](aclSetOutputTensorAddr.md).|
| AclSetDynamicInputTensorAddr | Has the same function as [aclSetDynamicInputTensorAddr](aclSetDynamicInputTensorAddr.md).|
| AclSetDynamicOutputTensorAddr | Has the same function as [aclSetDynamicOutputTensorAddr](aclSetDynamicOutputTensorAddr.md).|
| AclSetTensorAddr | Has the same function as [aclSetTensorAddr](aclSetTensorAddr.md).|
| AclSetDynamicTensorAddr | Has the same function as [aclSetDynamicTensorAddr](aclSetDynamicTensorAddr.md).|
